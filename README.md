# Читалища в България (1980–2000)

Интерактивно Streamlit приложение за визуализация на данни за читалищата в България по области за периода 1980–2000 г.

## Онлайн приложение

Приложението е достъпно онлайн тук:  
https://chitalishta-app.streamlit.app/

## Описание

Приложението позволява да се изследват:

- броят на читалищата по области (общо, в градове и в села) по години;
- броят на читателите (град/село);
- дисбалансът между читалищата в градовете и селата;
- динамиката във времето на национално ниво и на ниво 8 макрорегиона.

Визуализациите отчитат административно-териториалната реформа от 1987 г., при която 28-те окръга са реорганизирани в 8 области и Столична община.

## Данни

Данните са събрани и структурирани на база годишници на НСИ и други статистически публикации за периода 1980–2000 г.  
Основният файл е:

- `data_by_year.csv` – таблични данни по години и области (код и име на област, общ брой читалища, читалища в градове и села, читатели).

Геоданни:

- `provinces.geojson` – геометрия на 28 окръга;
- `provinces_9.geojson` – геометрия, агрегирана до 8 области (макрорегиони).

## Локално стартиране

1. Клонирай репото:

   ```bash
   git clone https://github.com/poli-petrova/chitalishta-streamlit.git
   cd chitalishta-streamlit

2. Създай и активирай виртуална среда (по избор):

  ```bash
  python -m venv .venv
  source .venv/bin/activate


3. Инсталирай зависимостите:

  ```bash
  pip install -r requirements.txt

4. Стартирай приложението:

  ```bash
 streamlit run app.py
  
След стартиране Streamlit ще отвори приложението в браузъра (обикновено на http://localhost:8501).
